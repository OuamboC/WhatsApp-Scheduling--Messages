## Troubleshooting 

Ensure that WhatsApp Web is logged in and open.

Verify that the phone number format is correct.

Adjust the delay (time.sleep) if the message remains in draft.

## Requirements

- Python 3.x
- `pywhatkit` library
- `pyautogui` library
- `pygetwindow` library

## Installation
